## 📱 Deskripsi Aplikasi

MyTaskList adalah aplikasi sederhana untuk mencatat kegiatan sehari-hari.
Pengguna dapat menambahkan, menghapus, dan menandai kegiatan sebagai selesai.
Aplikasi ini juga dilengkapi dengan fitur prioritas (Segera, Nanti, Bebas) untuk membantu mengatur kegiatan.

---

## ✨ Fitur Aplikasi

* [x] Menambahkan kegiatan
* [x] Menghapus kegiatan
* [x] Menandai kegiatan sebagai selesai
* [x] Validasi input (tidak boleh kosong)
* [x] Menampilkan daftar menggunakan FlatList
* [x] Empty state saat tidak ada data
* [x] Counter jumlah kegiatan selesai
* [x] Prioritas kegiatan (Segera / Nanti / Bebas)
* [x] Tampilan UI tema pink modern
